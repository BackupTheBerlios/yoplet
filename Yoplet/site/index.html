Yoplet

Once upon a time, WEB 2.0 was born. Revolution in process... at last!
But what about interaction with client system's resources? Any way to get a simple file content to be read/written ?

Yoplet is a free framework applet-based which provides commands execution for any client browser,
against any back-end os. it aims to simply realizes the commands/configuration patterns at an applet level
and comes with a bunch of integration tools (web server independant install, security issues, and more).

Currently migrating sources from private repository, and setting up installer.
